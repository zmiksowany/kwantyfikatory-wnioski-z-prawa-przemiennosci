Wnioski z prawa przemienności:

  • pierwsze dwa prawa pozwalają na wprowadzenie krótszego zapisu:

    • – ∃x, y : p(x, y) ≡ ∃x∃y : p(x, y)

    – ∀x, y : p(x, y) ≡ ∀x∀y : p(x, y)

  • trzecie prawo zapisane jest przy pomocy implikacji, co oznacza, że nie wolno nam zamieniać miejscami kwantyfikatora ∀ z kwantyfikatorem ∃, gdy to kwantyfikator ∀ stoi na pierwszym miejscu.

Kontrprzykład (w nawiązaniu do trzeciego prawa):

Rozważamy funkcję zdaniową x = y, dla której dziedziną są pary liczb rzeczywistych. Zdanie ∀y ∃x : x = y jest prawdziwe, gdyż dla dowolnej liczby rzeczywistej y możemy wskazać liczbę rzeczywistą x (tj. x := y) spełniającą zadany warunek. Natomiast zdanie ∃x ∀y : x = y jest fałszywe (bo dla każdej liczby rzeczywistej istnieje liczba od niej różna, np. większa o jeden). Zatem nie jest prawdziwa implikacja: (∀y ∃x : x = y) ⇒ (∃x ∀y : x = y).
